2. Hosting bei GitHub Pages

Diese Website wird über GitHub Pages ausgeliefert, einem Dienst der GitHub, Inc., 88 Colin P Kelly Jr Street, San Francisco, CA 94107, USA (eine Tochtergesellschaft der Microsoft Corporation).

Beim Aufruf der Seite werden durch GitHub aus technischen Gründen automatisch Daten verarbeitet, die dein Browser übermittelt:

  • IP-Adresse
  • Datum und Uhrzeit der Anfrage
  • Abgerufene URL und HTTP-Statuscode
  • Übertragene Datenmenge
  • Referrer und User-Agent (Browser, Betriebssystem)

Diese Daten sind erforderlich, um die Website ausliefern und ihre Stabilität sicherstellen zu können. Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an einer funktionsfähigen, sicheren Website).

Datenübermittlung in die USA

Da GitHub seinen Sitz in den USA hat, kann eine Übermittlung personenbezogener Daten in ein Drittland stattfinden. GitHub stellt für solche Übermittlungen Garantien nach Art. 46 DSGVO bereit (Standardvertragsklauseln); soweit GitHub am EU-US Data Privacy Framework teilnimmt, gilt zusätzlich der Angemessenheitsbeschluss der EU-Kommission nach Art. 45 DSGVO. Details zur Verarbeitung, Speicherdauer und den Garantien findest du in der Datenschutzerklärung von GitHub.

3. Schriftarten (Bunny Fonts)

Diese Seite bindet die Schriftart Atkinson Hyperlegible über Bunny Fonts ein, einen Dienst der BunnyWay d.o.o. mit Sitz in Slowenien (EU). Beim Aufruf der Seite stellt dein Browser eine Verbindung zu fonts.bunny.net her, wobei die IP-Adresse an den Dienst übermittelt wird. Laut Anbieter werden dabei keine IP-Adressen gespeichert und keine Cookies gesetzt. Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an einer einheitlichen, barrierefreundlichen Typografie).

4. Lokale Speicherung in der Anwendung

Die Anwendung unter /app/ speichert deine Eingaben (z. B. Absender, Empfänger, Brieftext) ausschließlich im lokalen Speicher deines Browsers (localStorage), damit du sie beim nächsten Besuch nicht erneut eingeben musst. Diese Daten werden zu keinem Zeitpunkt an einen Server übertragen und können jederzeit über die Einstellungen deines Browsers gelöscht werden.

Da diese Speicherung für die vom Nutzer ausdrücklich gewünschte Funktion (Verfassen eines Briefes) unbedingt erforderlich ist, ist sie nach §25 Abs. 2 Nr. 2 TDDDG ohne Einwilligung zulässig.

5. Kein Tracking, keine Analyse, keine Werbung

briefsatz.de setzt keine Cookies, keine Analyse-Werkzeuge und keine Werbe-Tracker ein. Es findet keine Profilbildung und keine Weitergabe von Daten an Dritte zu Werbezwecken statt.
